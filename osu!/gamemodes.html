Game Modes

In the game modes option, there are play, edit, options, and exit.

In the play section, there are solo and multi modes.

Types of Game Modes

There are 4 types of game modes: osu!, osu!mania, osu!catch, osu!taiko.

Solo Mode

In solo mode, there are songs selection, game mods, and editing songs.

Multi Mode

In multi mode, there is a lobby that contains online users and chat box for online social interation.

In the multiplayer lobby, you can join/create a multiplayer room for playing with friends.
